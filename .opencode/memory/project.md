# Project: caply

## Purpose
A Flutter mobile application.

## Tech Stack
- **Framework**: Flutter (Dart)
- **State Management**: Provider
- **Networking**: http package
- **Ads**: Google Mobile Ads
- **Local Storage**: shared_preferences
- **Icons**: cupertino_icons, font_awesome_flutter
- **Fonts**: google_fonts

## Architecture
Feature-first structure with shared core utilities.
- `lib/core/` — Theme, constants, extensions, shared widgets
- `lib/features/` — Feature modules, each with its own models/views/controllers

## Build Targets
Android, iOS, Web, Windows, macOS, Linux
